# Formulário de Matrícula - Rocketseat

## Sobre o Projeto

Este projeto foi desenvolvido durante a trilha da Rocketseat. Trata-se de um formulário de matrícula escolar focado em HTML semântico e estilização moderna, exercitando a organização de dados complexos em grupos e a melhoria da experiência do usuário.

## Tecnologias Utilizadas

- HTML5: Estrutura semântica e acessibilidade.
- CSS3: Layout com Flexbox, Grid e uso de variáveis.
- Google Fonts: Tipografia personalizada.

## Como Rodar e Visualizar o Projeto

1. Clone o repositório:
   git clone https://github.com/willianpm/Projeto-Rocketseat-Formulario_de_Matricula.git

2. Acesse a pasta:
   cd Projeto-Rocketseat-Formulario_de_Matricula

3. Abra o projeto:
   Basta abrir o arquivo index.html em qualquer navegador.
